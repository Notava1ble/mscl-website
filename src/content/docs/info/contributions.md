---

Ranked Leagues is a community-driven project! If you'd like to help out with the event, check out the `event-planner` role in the discord server. We're always looking for new people to help out with the event.

If you'd like to help out with the website, check out the [GitHub repository](https://github.com/Notava1ble/mscl-website).
